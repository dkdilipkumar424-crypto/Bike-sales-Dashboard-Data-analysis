# Bike-sales-Dashboard-Data-analysis
This project is an interactive Excel dashboard designed to explore who buys bikes and why. Using customer demographics, income, and commute data, the dashboard lets you quickly spot trends and compare different groups.
 What the Dashboard Shows

Average Salary of customers who purchased a bike vs. those who didnâ€™t

Purchases by Age Group, split by gender

Commute Distance Patterns and how they relate to buying behavior

Interactive Filters for home ownership, education, region, gender, and more

Key Takeaways

Higher income seems to increase the likelihood of buying a bike

Gender and age groups show different buying trends

Shorter commute distances often correlate with more purchases

Tools Used

Excel

Pivot Tables & Pivot Charts

Slicers for interactivity

How to Use

Download the Excel file, open it in Excel, and use the slicers on the left to explore the data.
